# Credit Card Analytics

## Project Description
This project analyzes credit card transaction data using Python and Excel. 
The dataset is generated using a Python script and then analyzed to understand transaction patterns, spending behavior, and potential fraud activities.

## Problem Statement
Financial institutions handle thousands of credit card transactions every day. 
Analyzing these transactions helps identify unusual patterns, monitor spending behavior, and detect possible fraudulent activities.

## Dataset
The dataset contains simulated credit card transactions including:
- Transaction ID
- Transaction date
- Customer information
- Transaction amount
- Country
- Transaction category
- Fraud flag

## Project Files
- transaction_processing.py – Python script used to generate and process transaction data
- transactions.csv – Generated dataset containing credit card transaction records
- credit_card_transaction_dashboard.xlsx – Excel dashboard used to visualize transaction insights

## Analysis Performed
The project includes analysis such as:
- Fraud vs Non-Fraud transaction distribution
- Spending analysis by category
- Country-wise transaction analysis
- Total transaction amount insights
